# Training Studio Website

This is a responsive website for a fictional gym named **Training Studio**. The website is designed using HTML, CSS, and JavaScript, and it features a variety of sections including a video background header, navigation links, program selection, class schedules, expert trainers, and a contact form with a Google Maps embed.

## Features

- **Responsive Design**: The website is fully responsive and works well on different screen sizes.
- **Video Background**: The header section features an autoplay video background to create a dynamic and engaging first impression.
- **Navigation Bar**: Includes links to different sections of the website such as Home, About, Classes, Schedules, and Contact.
- **Program Selection**: A section displaying various fitness programs with call-to-action buttons.
- **Class Schedules**: A table displaying class schedules with information about the fitness classes, timings, and trainers.
- **Expert Trainers**: A section showcasing the expert trainers with their bios and social media links.
- **Contact Form**: A form where users can send messages, along with an embedded Google Map showing the gym's location.
